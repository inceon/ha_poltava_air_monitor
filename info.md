# Poltava Air Monitor

[![GitHub Release](https://img.shields.io/github/release/inceon/ha-poltava-air-monitor.svg)](https://github.com/inceon/ha-poltava-air-monitor/releases)

Monitor air quality in Poltava, Ukraine directly in Home Assistant!

## Features

✅ Real-time air quality data from official Poltava monitoring stations  
✅ Multiple pollutants: PM2.5, PM10, PM1, O₃, NO₂, SO₂, CO  
✅ Weather data: Temperature, Humidity, Pressure, Wind  
✅ Air Quality Index with health recommendations  
✅ Choose from multiple monitoring stations or find nearest by coordinates  
✅ Updates every 5 minutes  
✅ Bilingual support (English/Ukrainian)

## Quick Start

1. Install the integration via HACS
2. Restart Home Assistant
3. Go to Settings → Devices & Services → Add Integration
4. Search for "Poltava Air Monitor"
5. Follow the configuration wizard

## Available Sensors

- Air Quality Index (AQI)
- PM2.5, PM10, PM1
- Ozone, Nitrogen Dioxide, Sulfur Dioxide, Carbon Monoxide
- Temperature, Humidity, Pressure
- Wind Speed & Direction

## Data Source

Data is provided by the official Poltava City air quality monitoring system at `improvement-pl.gov.ua`.
